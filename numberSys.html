INTRODUCTION

Number System these are the technique to represent numbers in the vcomputer system architecture and every data that you take and put in the memory of computer has equivalent number system

There are four types of Number System:

  1. BINARY
  2. OCTAL
  3. DECIMAL
  4. HEXADECIMAL

Each bases of Number System write as a subscript of the Number System like the example below;

  1. BINARY = BINARY02
  2. OCTAL = OCTAL08
  3. DECIMAL = OCTAL10
  4. HEXADECIMAL = OCTAL16

Each of Number System has equivalent bases 

    Binary is equivaleng to Base-6
    Octal is equivaleng to Base-8
    Decimal is equivaleng to Base-10
    Hexadecimal is equivaleng to Base-16

Each Number System has it own value and you can convert it to each other. This is the function of this website.
Look at the demo below;

  • STEP 1: the left side drop down box will be the number system you want to convert.
  • STEP 2: the right side drop down box will be the number system you want your value to be converted.
  • STEP 3: type the value that you want to be converted. if binary only accepts 0 and 1 value, if octal only accept from 0-7 value, if decimal only accept 0-9 and if hexadecimal it only accept 0-9 value and A,B,C,D,E,F hex value.
  • STEP 4: clink the convert button and wait for the result.
